# ctx — Context Handoff (VS Code Extension)

> 🚧 **Status:** This extension is under active development. v1.0.0 has not shipped yet.

Export and import AI-assisted development session context as portable `.ctx` bundles — a native VS Code experience wrapped around the [`ctx`](https://github.com/RiddhiKatarki/ctx) CLI.

See the [main project README](https://github.com/RiddhiKatarki/ctx) for the bundle format and CLI feature set. This README will be expanded with extension-specific usage, screenshots, and settings reference before the Marketplace release.
