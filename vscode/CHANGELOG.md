## [Unreleased]

### Added
- Initial VS Code extension scaffolding.
- Activity-bar view container with placeholder "Context Bundles" tree view.
- Command palette registrations for `export`, `import`, `inspect`, `info`, `applyPatch`, and `refresh`.
- Explorer context menu contributions for `*.ctx` files.
- Extension configuration schema covering summary provider, OpenAI settings, secret-scan toggles, content embedding, and notification verbosity.
